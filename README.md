# Hansini Rathnayaka | UI/UX Portfolio

This is a modern, responsive portfolio website for Hansini Rathnayaka, an Associate UI/UX Engineer. Built with React, Tailwind CSS, and Framer Motion for a premium digital experience.

## Features
- ✨ Modern, sleek design with glassmorphism effects
- 📱 Fully responsive across all devices (Desktop, Tablet, Mobile)
- 🚀 Smooth animations and transitions
- 🛠️ Comprehensive showcase of skills, experience, and projects


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
